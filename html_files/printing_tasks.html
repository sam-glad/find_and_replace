A task is a "to do" or maintenance item for your business.

Synonyms include request, reminder, and busy work.

For example:

  • Remember to schedule the some work to turn over a unit.
  • Run reports for your board or rental owners.
  • Get some milk on the way home.

Buildium allows you to classify tasks in a few ways:

To do A task created by staff and not categorized as another task type.
Resident request A task created by a tenant or association owner through their site. These can be created staff, too.
Rental owner request A task created by a rental owner through their site. These can be created staff, too.
Prospect request A task created by a visitor to your public website.

To print a task:

  1. Go to the Tasks menu.
  2. Go to the All tasks page.
  3. Click the taskyou want to print.
  4. Click .

  5. Select your desired level of detail and click Create PDF.

You can print many tasks at once by filtering the All tasks page to the tasks you want and clicking Print tasks.
