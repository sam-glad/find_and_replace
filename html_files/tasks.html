A task is a "to do" or maintenance item for your business.

Synonyms include request, reminder, and busy work.

For example:

  • Remember to schedule the some work to turn over a unit.
  • Run reports for your board or rental owners.
  • Get some milk on the way home.

Buildium allows you to classify tasks in a few ways:

   
To do A task created by staff and not categorized as another task type.
Resident request A task created by a tenant or association owner through their site. These can be created staff, too.
Rental owner request A task created by a rental owner through their site. These can be created staff, too.
Prospect request A task created by a visitor to your public website.

 

Save time! We've got an email notification about this feature that you may find quite convenient.

Add a task

Save time! Access this feature from any screen using the shortcuts menu.

  1. Go to the Tasks menu.
  2. Go to the My tasks page, the Unassigned tasks page or the All tasks page.

  3. Click Add task. Select the type of task that you wish to create.

  4. Complete the form.

    Save time! Use email notifications and the due date field to set up a reminder and stay on top of your work.

    Resident requests, rental owner requests, and prospect requests allow you to enter the contact information of the person making the request.

  5. Click Create task.

Change a task

  1. Go to the Tasks menu.
  2. Go to the All tasks page.
  3. Click the task that you want to change.

  4. Click .

  5. Complete the form.

  6. Click Save.

Update and assign a task

  1. Go to the Tasks menu.
  2. Go to the All tasks page.
  3. Click the task that you want to update or assign.

  4. Click Update task.

  5. Complete the form. 

    Resident requests and rental owner requests both allow you to choose to whether your update should be shared.

  6. Click Save. The updated information appears in the task history section below the task status. 

Once the update is made, you will have 5 minutes to select to Undo the task update. By selecting to Undo, you have the option to edit the update or delete the update.

Update a task's history

  1. Go to the Tasks menu.
  2. Go to the All tasks page.
  3. Click the task that you want to update.

  4. Click for the task history that needs to be changed.

  5. Complete the form.

  6. Click Save.

    Buildium will note the last time the history was changed and who made the change.

Delete a task

Once a task is done, consider changing its status to complete instead of deleting it. This way, you'll have a record of the task in case you need it in the future.

  1. Go to the Tasks menu.
  2. Go to the All tasks page.
  3. Click the task you want to delete.

  4. Click  

  5. A confirmation pop-up appears, asking you to confirm that you want to delete the task item.

  6. Click Delete task.

Add a work order to a task

  1. Go to the Tasks menu.
  2. Go to the All tasks page.
  3. Click the task to which you'd like to add the work order.

  4. Click the Work Order tab.

    You can also add work orders without the task first. 

  5. Click Add work order.

  6. Complete the form.

    Task information, such as the request type, contact information, and subject will automatically be added into the work order.

  7. Click Create work order.
