A recurring task is a task that repeats and can occur at set periodic intervals

For example:

  • Monthly maintenance or landscaping work
  • An annual reminder to change the batteries in a smoke alarm
  • Prep work for a monthly meeting with the board of a condo association
  • Check the lights in the parking lot
  • Renew permits or licenses for elevators

Add a recurring task

  1. Go to the Tasks menu.
  2. Go to the Recurring tasks page. A list of recurring items is displayed.
  3. Click Add recurring task.

  4. Complete the form.

  5. Click or . Which save option should I choose?

Recurring tasks process through an overnight batch process that occurs once a day. If you need to add a task to work with now, add it manually.

Change a recurring task

  1. Go to the Tasks menu.
  2. Go to the Recurring tasks page.

  3. Click the recurring task item that you want to edit.

Delete a recurring task

  1. Go to the Tasks menu.
  2. Go to the Recurring tasks page.
  3. Click the next date corresponding to the recurring task item that you want to delete.

  5. A confirmation pop-up window prompts you to confirm the deletion. Click OK to delete the recurring task item.
