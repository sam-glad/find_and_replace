Article #: 112314

Use categories to organize resident site contacts, vendors, and tasks.

For example:

  • Add taxi numbers to your "taxi bucket" of phone numbers.
  • Organize and classify vendors by category.
  • View the Details column to determine the number of assigned numbers/vendors per category.
  • Identify some tasks as "maintenance", "feedback", or "general inquiry."

Add a vendor category

  1. Go to the Accounting menu.
  2. Go to the Vendors page.
  3. Click Manage categories.
  4. The Vendors Categories pop-up window appears, with a list of available categories.
  5. Click Add a new category.
  6. In the Category Name field that appears, enter the name of the category. Remember to use a name that is clear and unambiguous.
  7. Click the Add this category button to add the complaint. You can add as many categories as you want.
  8. When you are finished adding categories, click the Done Adding Categories link.
  9. Click to return to the Vendors page.

Change a vendor category

  1. Go to the Accounting menu.
  2. Go to the Vendors page.
  3. Click the Maintain Categories buttonabove the list of items.
  4. The Vendor Categories pop-up window appears, with a list of available categories.
  5. Under the Action column, click the Edit link corresponding to the category that you want to edit.
  6. In the Category Name field that appears, edit the name of the category.
  7. Click the Save changes button to save the new name of the category.
  8. Click to return to the All Vendors page.

Delete a vendor category

  1. Go to the Accounting menu.
  2. Go to the Vendors page.
  3. Click the Maintain Categories buttonabove the list of items.
  4. The Vendor Categories pop-up window appears, with a list of available categories.
  5. Under the Action column, click the Delete link corresponding to the category that you want to delete.
  6. A confirmation pop-up window prompts you to confirm the deletion. Click OK to delete the category.
  7. Click to return to the Vendors page.

When you delete a category, all items assigned to that category are marked Uncategorized.
